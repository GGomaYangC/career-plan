# career-plan

안녕하세요. 저는 컴퓨터공학과에 재학 중인 양승호입니다.  
현재 학교에서 C언어, Java, 자료구조, 알고리즘, 선형대수 등을 배우며 컴퓨터공학의 기초를 쌓고 있습니다.

제가 관심을 가지고 있는 분야는 인공지능과 소프트웨어 개발입니다.  
특히 인공지능이 데이터를 바탕으로 문제를 해결하고, 사람의 일을 도와주는 기술이라는 점에서 흥미를 느끼고 있습니다.

현재는 전공 기초를 탄탄하게 만들기 위해 프로그래밍 언어와 자료구조를 중심으로 공부하고 있습니다.  
C언어를 통해 기본적인 문법과 포인터, 자료구조의 원리를 익히고 있으며, Java를 통해 객체지향 프로그래밍 개념을 배우고 있습니다.

또한 인공지능 분야로 나아가기 위해서는 수학적 기초도 중요하다고 생각하여 선형대수와 알고리즘 공부도 함께 하고 있습니다.

앞으로는 Python을 더 공부하고, 인공지능과 머신러닝의 기본 개념을 익힐 계획입니다.  
단순히 이론만 공부하는 것이 아니라 간단한 프로젝트를 직접 만들어 보면서 실력을 키우고 싶습니다.

예를 들어 이미지 분류, 데이터 분석, 간단한 예측 프로그램 같은 프로젝트를 해보며 인공지능이 실제로 어떻게 활용되는지 경험해보고 싶습니다.

저의 최종 목표는 인공지능 기술을 이해하고 활용할 수 있는 개발자가 되는 것입니다.  
아직 부족한 부분이 많지만, 학교에서 배우는 전공 지식을 바탕으로 꾸준히 공부하면서 실력을 쌓아가겠습니다.
